3.5. Les dimensions exécutives impliquées dans le TDAH
Les fonctions les plus étudiées sont (Pasini et al., 2007; Sergeant et al., 2002; Solanto et al., 2007; Willcutt et al., 2005) :
•	l’inhibition
•	la mémoire de travail
•	la flexibilité cognitive
•	la planification
L’inhibition correspond à la capacité de résister ou supprimer une réponse non pertinente au but visé (Simpson & Riggs, 2007).
